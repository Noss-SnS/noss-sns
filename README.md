# AI Site Generator - مشروع إنشاء مواقع بالذكاء الاصطناعي (بدون مفتاح API)
هذا مشروع تجريبي (Scaffold) لموقع **يصنع موقعًا لك بالـ AI** بدون أن يحتاج المستخدم لمفتاح API. الفكرة: أنت تستضيف نموذج الاستدلال محليًا (مثلاً vLLM / Ollama / HF Inference Server)؛ الواجهة تتواصل مع خادمك مباشرة.

المحتوى:
- frontend/: واجهة HTML/CSS/JS بسيطة لجمع مواصفات الموقع (العنوان، نوع الموقع، الألوان).
- backend/: خادم Node.js (Express) يحتوي على endpoint `/generate` يتصل بخادم الاستدلال المحلي `http://localhost:8000/generate`. إذا لم يتوفر، فسيعود بخيار بديل يعتمد على قوالب محلية.
- templates/: قوالب HTML جاهزة يمكن للـ AI تخصيصها أو استخدامها كـ fallback.
- prompt_templates/: نصوص prompt جاهزة لاستخدامها مع نموذجك.

كيفية التشغيل (محلياً):
1. ثبت Node.js (الإصدار 18+).
2. افتح طرفية في المجلد `backend`:
   ```bash
   cd backend
   npm install
   node index.js
   ```
   الخادم سيشتغل على http://localhost:3000
3. افتح `frontend/index.html` في متصفحك أو استعمل سيرفر محلي لعرضه (Live Server).
4. لإكمال التكامل مع نموذج AI محلي، شغّل خادم الاستدلال لديك واستعمل المسار `http://localhost:8000/generate` لاستقبال POST مع الشكل:
   `{ "prompt": "...", "max_tokens": 2000 }`
   وتعيد استجابة JSON مثل: `{ "html": "<!doctype html>....", "css": "...", "assets": [] }`

ملاحظات:
- هذا مشروع قابل للتطوير: أضف طابور Redis، نظام دفع، تخزين S3، وworker منفصل عند الانتقال للإنتاج.
- تأكد من تراخيص النموذج المستخدم قبل التشغيل في بيئة تجارية.
